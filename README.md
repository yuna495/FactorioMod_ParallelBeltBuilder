# Parallel Belt Builder

Factorio 2.0に対応する、搬送ベルトの並列一括敷設Modです。
プレイヤーが手動で敷設した1本の「基準ベルト経路」を検出し、その左右に指定された本数だけ並列にベルト（実体またはゴースト）を自動配置します。

## 主な機能

* **並列本数の調整**: 2本〜16本の間で並列数を自由に設定可能。
* **ベルト種類の自動検知**: 黄色、赤色、青色など、最初に置いた搬送ベルトのプロトタイプを自動取得して複製に適用（他Modで追加された `transport-belt` 型ベルトにも対応）。
* **左右展開切り替え**: 進行方向の左側または右側への展開方向を設定。
* **通常配置とゴースト配置**:
  * **Normal**: プレイヤーのインベントリから必要数を消費して実体を配置（不足時は配置をキャンセルし、部分的な切断を防ぐ）。
  * **Ghost**: アイテム消費なしでロボット用のエンティティゴーストを配置。
* **曲がり角・複数曲折対応**: 直角（90度）の方向転換が含まれる経路に対応し、内側と外側の接続の長さを自動で補正・交差接続します。
* **障害物のスキップ**: 水や他の建物と衝突するタイルのみを自動でスキップし、配置可能な陸地のみに敷設。
* **エラーハンドリング**: 分岐、合流、ループ、180度折り返し、または内側経路が狭すぎて崩壊する形状を検知した場合は、配置を行わず警告を表示。
* **マルチプレイヤー＆セーブ・ロード対応**: プレイヤー個別のGUI設定や記録状態を安全に永続化。

## 使用方法

1. 画面下部またはショートカットメニューから **Parallel Belt Builder** ボタンをクリックして、設定GUIを開きます。
2. GUIで `Count` (本数)、`Side` (左右)、`Placement` (通常/ゴースト) を設定します（これらは記録開始前のみ変更可能です）。
3. **`Start recording`** ボタンをクリックして記録を開始します。
4. 通常通り、ゲーム内で搬送ベルトを一直線、あるいは曲がり角を含めて敷設します。
5. 敷設が終わったら、GUIの **`Stop recording`** ボタンをクリックします。
6. 基準経路の解析と接続計算が走り、即座に並列ベルトが敷設されます。
7. 記録を途中で破棄したい場合は **`Cancel`** ボタン（または `Esc` キー）を押します。
